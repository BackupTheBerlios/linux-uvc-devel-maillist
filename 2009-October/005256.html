> off"? Maybe given by an option on module load?
> 
> Bus 005 Device 002: ID 046d:09a5 Logitech, Inc.
> 
> I can switch the LED off with guvcview, of course. But i want to have
> it off without loading guvcview.
> 
> Is this possible? Thanks

There is no standard LED control in the UVC specification. Logitech webcams 
expose the LED control through so called extension units.

Extension units can be accessed through the V4L2 control API, but you first 
need to "configure" the driver to map extension units to V4L2 controls. The 
uvcdynctrl tool (part of libwebcam) available at 
http://www.quickcamteam.net/software/libwebcam/ can be used for that. It takes 
its configuration from an XML file provided in the software sources. Make sure 
you checkout the SVN version of libwebcam, the 0.1.1 tarball is outdated.
