>     * Close Skype
>     * Create directory "Logs" under your Skype data directory (~/.Skype/)
>     * Restart Skype
>     * Log file (skype_yyyy-mm-dd-hhmmss) will be created in ~/.Skype/Logs
>     * To turn off logging again, delete the "Logs" directory
>
