Hello,

as I hobby, I'm doing robotics research. So far, we've used an Orange
Micro iBot2 webcam on our robot, which has an OV7620 sensor connected to
USB 2.0 via a simple bridge implemented in the Cypress Semi EzUSB FX2
microprocessor. 

This setup allows us to get the raw unprocessed data from the OV7620 VGA
CMOS sensor in Bayer format - one byte per pixel, when the sensor is
programmed right (via its i2c registers). (We also were able to do full
VGA resolution at 60 fps, but I don't expect that'll be possible on any
other camera ...)

This saves us both USB and RAM bandwidth, because in Bayer mode we only
send 640x480 bytes, while YUYV mode needs 640x480x2 bytes.

Unfortunately, the iBot2 is no longer manufactured.

So the question is: Is it possible to get the raw data out of the
QuickCam Pro 5000 CCD VGA sensor?
