> Hi,
>
> http://www.everythingusb.com/logitech_quickcam_for_notebooks_pro.html
>
> This article mentions that the windows driver can enable "Low Light  
> Boost" which increases the exposure time in poor light conditions.
>
> 1. Does this feature exist in the linux-uvc driver ?
> 2. Does anybody know anything about this feature.

The feature is in the device itself so there's nothing special that the  
driver has to do about it except supporting the corresponding UVC control.  
Luckily, the uvcvideo driver does that with the help of the custom V4L2  
control 'Exposure, auto' (V4L2_CID_EXPOSURE_AUTO). So if you have an  
application that can toggle this control, it should work.
