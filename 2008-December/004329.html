On Sunday 07 December 2008, Sven Burmeister wrote:
> Hello!
>
> I used to compile from berlios. I got the mercurial repo, so should I
> compile only linux uvc, i.e. uvcvideo/linux/drivers/media/video/uvc , or
> should i compile and install everything via make all?

You need to compile and install (or load manually) at least the following 
modules

v4l1-compat.ko
v4l2-compat-ioctl32.ko
videodev.ko
uvcvideo.ko

The easiest way is to compile the whole tree and either install all modules 
(using make install) or copy or load the above four ones manually. Please 
note that v4l drivers shipped with your kernel will not work with the 
videodev.ko module from the mercurial tree.
