Hi Bob,

On Wednesday 03 December 2008, Robert Lewis wrote:
> How do I adjust the color correction for my Logitech 9000 so that Skype on
> Linux looks right?
>
> Currently white walls, etc. shows up with a yellow tint because of
> florescent lighting?

The camera implements a white balance correction algorithm that should help 
you. This can be controlled from any V4L2 userspace application much like 
other controls (brightness, contrast, ...). If Skype doesn't expose the white 
balance control, use another application (luvcview and ucview come to mind 
but there are plenty of other applications, even command line based such as 
uvcdynctrl). The control won't be modified when you close the application 
(unless Skype explicitly reverts it to another value).
